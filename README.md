# Retail Sales & Profitability Analysis Dashboard

## Project Overview
This project analyzes retail sales data to understand business performance, profitability drivers, and the impact of discount strategies. The analysis combines **Python-based Exploratory Data Analysis (EDA)** with an **interactive Power BI dashboard** to uncover actionable business insights.

The goal is to move beyond simple reporting and identify patterns that can help businesses optimize pricing strategies, improve profitability, and monitor operational efficiency.

---

## Business Questions
This analysis aims to answer key business questions:

- Which products generate the highest profit?
- Which products are causing financial losses?
- How do discounts affect profitability?
- Which regions generate the highest sales?
- How efficient is the delivery process?

---

## Dataset
Dataset used: **Sample Superstore Dataset**

Key columns include:

- Order ID
- Order Date
- Ship Date
- Sales
- Profit
- Discount
- Category
- Sub-Category
- Region
- Customer Segment

---

## Data Cleaning
Data preprocessing steps included:

- Handling inconsistent date formats
- Converting date columns to proper date format
- Checking missing values
- Removing duplicate records
- Validating numeric fields such as sales, profit, and discount

Tools used: **Python (Pandas) and Excel**

## Exploratory Data Analysis (Python)

Exploratory Data Analysis was performed using Python to identify patterns and relationships in the dataset.

Libraries used:

- Pandas
- Matplotlib
- Seaborn

EDA focused on:

- Sales distribution
- Profit trends
- Discount impact on profit

---

## Key Business Insights

**1. Discount Impact on Profit**  
Higher discounts significantly reduce profit margins in several product sub-categories.

**2. Loss-Making Products**  
Certain products generate negative profit and may require pricing or discount adjustments.

**3. Category Performance**  
The Furniture category shows significantly lower profit margins compared to Technology and Office Supplies.

**4. Operational Efficiency**  
Average delivery time helps evaluate shipping and logistics performance.
